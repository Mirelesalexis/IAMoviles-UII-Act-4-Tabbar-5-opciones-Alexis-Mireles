Prompt para Generación de App Flutter: "Domino's TabBar con Rutas"
"Actúa como un desarrollador experto en Flutter. Genera el código completo para una aplicación de Domino's Pizza siguiendo estas especificaciones estrictas:

Estructura de Carpetas: Todos los archivos de las pantallas deben estar en una carpeta llamada lib/mis_pantallas_tab/.

Navegación Principal: Implementa un Scaffold con un BottomNavigationBar (TabBar inferior) que contenga 5 opciones: Inicio (Icons.home), Pizzas (Icons.local_pizza), Bebidas (Icons.local_drink), Más Comida (Icons.fastfood) y Tiendas (Icons.storefront).

Rutas Nombradas: En el archivo main.dart, configura initialRoute: '/' y define el mapa de routes para: '/', '/inicio', '/pizzas', '/bebidas', '/mas_comida' y '/sucursales'.

Pantallas Individuales: Crea 5 archivos .dart independientes dentro de la carpeta mencionada. Cada pantalla debe tener:

Un AppBar con los colores corporativos de Domino's (Rojo: #E31837 o Azul: #006491).

Un título centralizado acorde a la sección.

Una imagen proveniente de una URL de la red (GitHub o similar) usando Image.network con un loadingBuilder.

Texto descriptivo debajo de la imagen.

Lógica de Inicio: Configura la App para que al abrirse la primera pantalla visible sea la de Inicio (índice 0 del TabBar).

Estilo: Usa BottomNavigationBarType.fixed para que todos los labels sean visibles y aplica un diseño limpio y profesional."
